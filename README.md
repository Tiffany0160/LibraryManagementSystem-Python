# LibraryManagementSystem-Python

## Description
A console-based Library Management System built with Python and SQLite.  
Features:
- Add books
- View all books
- Delete books
- Borrow and return books

## Built With
- Python 3
- SQLite3
- PrettyTable (for console table formatting)
- PyCharm IDE

## How to Run
1. Clone the repository.
2. Open in PyCharm.
3. Run `main.py`.
4. Use the menu to manage library books.
